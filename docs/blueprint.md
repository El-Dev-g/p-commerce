# **App Name**: Curated Finds

## Core Features:

- Product Catalog: Display products with prices, descriptions, and images in an organized and searchable catalog.
- Shopping Cart: Allow customers to add selected products to a shopping cart and adjust quantities.
- AI-Powered Product Recommendations: An AI tool analyzes browsing history to recommend related products, enhancing the shopping experience.

## Style Guidelines:

- Primary color: Deep Indigo (#4B0082) to evoke trust and sophistication.
- Background color: Light Lavender (#E6E6FA) for a soft, inviting feel.
- Accent color: Violet (#8F00FF) for highlights and calls to action, providing contrast.
- Body font: 'PT Sans', a versatile and readable sans-serif.
- Headline font: 'Playfair', a modern sans-serif to complement the main body of text. Use 'PT Sans' for longer paragraphs.
- Clean, minimalist icons for navigation and product categories to provide easy understanding.
- Grid-based layout with clear spacing for a visually appealing and responsive design.
- Subtle animations and transitions on hover and during navigation to enhance user engagement.
